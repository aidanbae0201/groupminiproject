히딩크 대여 시간 관리 프로그램
히딩크 필드 대여를 스프레드시트가 아닌 CRUD 프로그램으로 구현해보고자 했습니다.
기능들:
- 날짜와 시간, 인원수, 학번을 입력받아 해당 날짜와 시간에 자리 예약.
- 학번을 검색해서 대여한 날짜와 시간, 인원수 파악 기능
- 인원수 검색 기능을 통해서 부족한 인원 충당 가능성
- 날짜 검색해서 그 날 예약돼어 있는 시간과 인원수 파악 기능
